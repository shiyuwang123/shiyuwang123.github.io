# My Personal Blog

Welcome to my personal blog! This project is a minimalist, Apple-inspired static website built with Next.js, TypeScript, and TailwindCSS.

## Features
- **Next.js App Router**: Fast and modern React framework.
- **TypeScript**: Type-safe codebase.
- **TailwindCSS**: Utility-first styling with an Apple-like aesthetic.
- **Rich Content**: Supports Markdown-like features natively in React, including KaTeX math formulas and syntax-highlighted code blocks.
- **Static Export**: Fully compatible with GitHub Pages.

## Getting Started
To run the project locally:
1. Install dependencies: `bun install`
2. Start the development server: `bun run dev`
3. Open `http://localhost:3000` in your browser.

For detailed documentation on how to write posts and use the site's APIs, please visit the `/docs` page on the website.
